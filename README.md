Welcome to the bua project! bua is an AI-powered browser automation tool built in Go. It allows you to describe tasks in plain English. The agent then takes care of the clicks and actions for you.

- **Web Scraping Made Simple**: Extract data from websites effortlessly.

1. **Open bua**: Double-click the icon on your desktop or in your application folder.
